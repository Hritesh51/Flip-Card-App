# Flip Card App

A small front-end project featuring an interactive flip-card UI and a contact page, built with HTML and CSS. It was created to practice CSS transforms, transitions, and hover-based animations.

## Contents

`flipcard51.html` contains the flip-card animation, and `contact51.html` contains a simple contact page.

## Tech Stack

HTML and CSS

## Getting Started

Clone the repository and open either HTML file directly in your browser.

## Status

A learning project built while exploring CSS animations and layout.
